## Frontend - To-Do List App (React + Vite)

This is the **frontend** of the MERN To-Do List App, built using **React**, **Vite**, **Toastify** and **Axios**. It connects to the backend API to perform CRUD operations on tasks.

---

## Tech Stack

- React + Vite
- Axios
- React Toastify

---

## Dependencies 

  "axios"
  "react"
  "react-dom"
  "react-router-dom"
  "react-toastify"
